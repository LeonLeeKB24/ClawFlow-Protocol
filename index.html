Not a platform.
A protocol.

Zapier and n8n are tools for humans to build workflows visually. ClawFlow is a standard for how AI agents discover and call those workflows programmatically.

Any workflow published in ClawFlow format is instantly queryable by any AI agent. The protocol — not the platform — is what creates network effects.

First to provide cost + success rate data per workflow becomes the default source for agent-driven task execution.

// for developers

Publish your workflow in 3 steps

STEP 01

Define the schema

Specify input/output format, estimated token cost, and add your endpoint URL.

STEP 02

Submit to registry

POST your workflow JSON to the ClawFlow API. Automatic validation and benchmarking runs within 24h.

STEP 03

Agents find you

Once listed, any agent querying ClawFlow for your task type can discover and call your workflow directly.
